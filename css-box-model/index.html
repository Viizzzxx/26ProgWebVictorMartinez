Parte 3 — Observación

Parece que deben caber 3 tarjetas, pero la tercera se baja porque también cuentan el padding y el borde.

Parte 4 — Análisis matemático

Cada tarjeta no mide solo 300px. También se suman 40px de padding y 20px de borde, así que en total mide 360px.

Parte 5 — Experimento

Al usar border-box, el ancho ya incluye el borde y el padding.

Parte 6 — Observación nuevamente

Ahora las 3 tarjetas sí caben mejor porque el ancho ya no crece de más.

Parte 7 — Experimento avanzado

Con porcentajes también puede romperse el diseño si no se usa border-box. Con border-box funciona mejor.

Parte 8 — Experimento profesional (Flexbox)

Flexbox acomoda mejor las tarjetas, pero border-box sigue ayudando a que el diseño no se desborde.

Parte 9 — Experimento extremo

Si el padding y el borde aumentan mucho, sin border-box el diseño se rompe más fácil.

Parte 10 — Conclusión

Border-box ayuda a controlar mejor el ancho y por eso se usa mucho en CSS moderno.

Respuestas

  • Deben caber 3 tarjetas.
  • La tercera se baja por el padding y el borde.
  • Con border-box sí caben mejor.
  • Flexbox ayuda a ordenar.
  • Border-box hace el diseño más fácil.
